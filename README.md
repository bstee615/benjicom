# BenjiCom
My personal website and portfolio for employers and others interested.

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies frontend
```
yarn run build
```

### Watches frontend for compile
```
yarn run watch
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```
